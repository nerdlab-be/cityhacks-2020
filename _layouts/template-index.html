Open data en tools

Alles over bezoeken

  • Alle Brugse cultuurhuizen gebruiken hetzelfde systeem om bezoekers te registreren. Hier kunnen we welke voorstellingen of optredens mensen gaan bekijken zijn: van Concertgebouw over Stadschouwburg tot Biekorf.

    Daarnaast zijn alle voorstellingen die ooit in het concertgebouw hebben plaats gevonden proper opgelijst met metadata en al! {% include atom.icon.html value="external" %}
  • Publiq houdt in zijn UITdatabank alle culturele gebeurtenissen bij en daar is een API {% include atom.icon.html value="external" %} van. Vraag ons naar de Keys.
  • Iets doen met schilderijen? Geen probleem, alle wat in Groeninge hangt {% include atom.icon.html value="external" %} staat proper op WikiData.
  • Al deze zaligheden kunnen we koppelen aan de nabijheid van cultuurhuizen en leestempels {% include atom.icon.html value="external" %}.

Bouw het zelf

  • Goesting om een nieuwe dataset op te bouwen? Dat mag ook. Als je aan de slag gaat met sensoren om beleving te meten is het goed dat de data achteraf weer bruikbaar is. Duidelijke structuur, gekuiste info en met goeie metadata.

Meer hierover in de slides van Pieter Colpaert {% include atom.icon.html value="external" %}

Nog relevante data

Binnen verschillende thema's heeft Vlaanderen maar ook de regio rond Brugge verschillende datasets beschikbaar. Gewoon snoepen.

  • Open data portaal EU {% include atom.icon.html value="external" %}: meer dan 10k datasets over alles wat je op europees niveau wil bijhouden.
  • Data.gov {% include atom.icon.html value="external" %}: 6500 datasets in allerlei onderwerpen op nationaal niveau
  • Kortrijksanen krijgen cultuurtips op maat, een onderzoek door Cultuurconnect {% include atom.icon.html value="external" %}. Lezen die handel
  • Leestipper {% include atom.icon.html value="external" %}: een soort discover weekly voor boeken:
